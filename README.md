# Rust For Visual Studio Code

This is a work in progress extension to Visual Studio Code that aims sto be a light-weight integration of various IDE-tools for Rust.

## Features

- racer
    - Go to definition
    - Peek definition
    - Hover definition
    - Signature help
    - Auto complete
- rustfmt
    - Format file
- other
    - Create tasks.json
        - build
        - doc
        - fmt
        - run
        - test
